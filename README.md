# Backend

Node.js + Express + MongoDB API for Ethical.

## Folders
- `controllers/` — Route logic
- `middleware/` — Auth, error, etc.
- `models/` — Mongoose schemas
- `routes/` — Express routers
- `utils/` — Helpers, rule engine

## Setup
1. `npm install`
2. Create `.env` (see `.env.example`)
3. `npm start`
